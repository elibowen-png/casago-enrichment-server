This tool needs a backend server to look up contact info. Paste the URL of your deployed server below. To get a free server URL, deploy to Render.com using the files in this package.

How to deploy the server (one-time setup):
1. Create a free account at render.com
2. Upload pm_enrichment_server.py, requirements.txt, and Procfile to a GitHub repo
3. In Render → New → Web Service → connect the repo → Deploy
4. Copy the https://your-app.onrender.com URL and paste it below
5. Host pm_prospecting_dashboard.html on Netlify Drop (free, drag & drop)
